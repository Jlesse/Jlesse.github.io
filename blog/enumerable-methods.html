How To Group_by

via GIPHY

Enumerables are pretty effin cool huh? They are very useful and are packed full of useful methods for your programming pleasure. There are so many good ones, but today we are going to zero in on one in particular. The enumerable method group_by groups a collection by the result of the block. Group_by returns a hash with keys that are the evaluated result from the from the block, and each key is paired with an array of values the correspond to that key. Probably sounds a little confusing. Lets do some examples to see what really going on here.

We create a variable called first_letter enum. We assign it the value returned by breakfasts.group_by. In our group_by block, breakfast stands in for each string in the array as we iterate through it. Breakfast[0] tells the group_by method that we are grouping the words by the 0th index of each string AKA the first letter.

Hopefully these examples were helpful. you can see from the examples that group_by returns an enum that pairs each value with the key that is returned from the statement inside the code block. In the final example, all the strings that returned true for the include? method inside the block, are in one array and paired with the key true. All the strings that returned false for .include? are in another array and paired with false.
